ttk::entry::ExtendTo   w index  

ExtendTo – Extend the selection to the specified index.

The other end of the selection (the anchor) is determined as follows:

(1) if there is no selection, the anchor is the insert cursor; (2) if the index is outside the selection, grow the selection; (3) if the insert cursor is at one end of the selection, anchor the other end (4) otherwise anchor the start of the selection

The insert cursor is placed at the new end of the selection.

Returns: selection anchor.
