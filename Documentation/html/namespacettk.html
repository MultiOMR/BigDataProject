ttk::takesFocus   w  

ttk::takesFocus w – Test if the widget can take keyboard focus:

  • widget is viewable, AND:
  • if -takefocus is missing or empty, return 0, OR
  • if -takefocus is 0 or 1, return that value, OR
  • append the widget name to -takefocus and evaluate it as a script.

See also: tk::FocusOK

Note: This routine doesn't implement the same fallback heuristics as tk::FocusOK.
